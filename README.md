## Verify Setup
1. Validate Python version `python --version`
2. Validate pip `pip --version`

## Setup venv
0. `cd` into the repo
1. Create venv
`python -m venv myenv`
2. Activate venv
`myenv\Scripts\activate.bat`
3. Install dependencies
`python install -r requirements.txt`
